# HTML Slides

Place your slide files here before running the generator.

Each file should follow this format:

* `page_1.html`
* `page_2.html`
* `page_3.html`
* ...

Each HTML file represents **one slide**.

Make sure:

* Files are named in order
* Each slide is designed for 1280x720 resolution
* No scrolling (fixed layout)

After adding files, run:

```
npm run generate
```
